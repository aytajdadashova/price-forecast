## 📚 İstifadə Olunan Texnologiyalar
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## 🚀 Layihənin Gedişi
1. Dataset Hazırlığı
2. Data Təmizliyi və Encoding
3. Model Qurulması
4. Proqnoz və Qiymətləndirmə
5. Vizuallaşdırma
